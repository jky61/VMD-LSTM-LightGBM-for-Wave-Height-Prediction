# VMD-LSTM-LightGBM-for-Wave-Height-Prediction
This repository contains the core algorithmic implementation for the paper "A Synergistic VMD-LSTM-LightGBM Model for Significant Wave Height Prediction in the Northern Taiwan Strait". It provides a hybrid forecasting framework that uses Variational Mode Decomposition (VMD) to process non-stationary wave signals, PyTorch-based LSTM networks to extract temporal dependencies, and LightGBM to fit high-frequency stochastic residuals. Please note: To protect institutional data privacy and core intellectual property, the raw multi-station buoy datasets, specific optimal hyperparameter configurations, and custom data interpolation rules have been masked or removed. Researchers and engineers can use this robust structural template as a foundation to configure parameters and train the model on their own datasets.
